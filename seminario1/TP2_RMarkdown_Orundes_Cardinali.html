Introducción

Se trabaja con la base de datos Student0405.txt, que contiene datos recolectados durante los años 2004 y 2005 de estudiantes en clases de estadística de una Universidad de Estados Unidos.

Las variables consideradas son: género (Sex), promedio de calificaciones (GPA), importancia de la religión en sus vidas (ReligImp), cantidad de faltas promedio por semana (MissClass), ubicación en el aula (Seat), días de fiesta en un mes (PartyDays) y horas semanales de estudio (StudyHrs).

a) Visualización de la base de datos y variables

La primera alternativa de lectura con read.table() produjo un error porque la base está separada por tabulaciones. Por ese motivo se utiliza read.delim() indicando sep = "\t".

# Ruta del archivo de datos.
# Si el archivo está dentro de la carpeta seminario1, se usa esa ruta.
# Si está en la misma carpeta del Rmd, se usa Student0405.txt.
ruta_datos <- if (file.exists("seminario1/Student0405.txt")) {
  "seminario1/Student0405.txt"
} else {
  "Student0405.txt"
}

# Carga de datos separados por tabulaciones.
datos <- read.delim(
  ruta_datos,
  sep = "\t",
  header = TRUE,
  strip.white = TRUE,
  na.strings = c("", "NA")
)

# Conversión de variables numéricas.
datos$GPA <- as.numeric(datos$GPA)
datos$MissClass <- as.numeric(datos$MissClass)
datos$PartyDays <- as.numeric(datos$PartyDays)
datos$StudyHrs <- as.numeric(datos$StudyHrs)

# Limpieza de espacios y valores vacíos en Seat.
datos$Seat <- trimws(datos$Seat)
datos$Seat[datos$Seat == ""] <- NA

# Adjuntamos la base de datos.
attach(datos)

j) Prueba de hipótesis para comparar GPA promedio por género

Como no hay sospechas previas, corresponde una prueba bilateral.

Hipótesis nula:
\(H_0\): No existen diferencias entre el promedio de GPA de mujeres y varones. Es decir, \(\mu_1 = \mu_2\).

Hipótesis alternativa:
\(H_1\): Existen diferencias entre el promedio de GPA de mujeres y varones. Es decir, \(\mu_1 \neq \mu_2\).

Observaciones de Paolo Orundés Cardinali

En el grupo femenino se obtuvo un p-value de Shapiro-Wilk menor que 0,05 y en el grupo masculino también se obtuvo un p-value menor que 0,05. Por lo tanto, se rechaza la hipótesis de normalidad en ambos grupos según el test de Shapiro-Wilk.

Para analizar la homogeneidad de varianzas se aplicó el test de Levene. Como el p-value obtenido es mayor que 0,05, no se rechaza la hipótesis de igualdad de varianzas. Por lo tanto, puede considerarse que las varianzas de GPA entre mujeres y varones son homogéneas.

Luego se realizó una prueba t bilateral para comparar el promedio de GPA entre mujeres y varones. La hipótesis nula plantea que no existen diferencias entre los promedios de ambos grupos, mientras que la hipótesis alternativa plantea que sí existen diferencias.

Dado que el p-value obtenido en la prueba t es menor que 0,05, se rechaza la hipótesis nula. Por lo tanto, se concluye que existen diferencias estadísticamente significativas entre el promedio de GPA de mujeres y varones. A partir de los promedios muestrales, se observa que el grupo femenino presenta un GPA promedio mayor que el grupo masculino.

Si bien el supuesto de normalidad no se cumple según Shapiro-Wilk, los tamaños muestrales son grandes en ambos grupos, por lo que la prueba t puede considerarse robusta frente a desviaciones de normalidad.

k) Intervalo de confianza del 98% para la verdadera calificación media

La consigna indica que este intervalo debe calcularse solo si no existen diferencias significativas del promedio de calificaciones según género.

A partir de la prueba t realizada en el inciso j), se obtuvo un p-value menor que 0,05. Por lo tanto, se concluyó que existen diferencias estadísticamente significativas entre los promedios de GPA de mujeres y varones.

Observaciones de Paolo Orundés Cardinali

No se desarrolla el cálculo de un intervalo de confianza general, dado que la condición inicial no se cumple. La prueba t realizada en el inciso j) mostró diferencias significativas entre los promedios de GPA según género. Por ello, no resulta adecuado estimar una única media poblacional para todos los estudiantes sin distinguir entre mujeres y varones.
